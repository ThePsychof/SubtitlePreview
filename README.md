# SubtitlePreview

Preview subtitle styles in a realistic movie frame before hardsubbing.

SubtitlePreview is a lightweight browser-based tool that lets clients and subtitle editors preview subtitle appearance without rendering an entire video.

No installation.
No account.
Runs entirely in your browser.

---

## Features

- 🎬 Realistic movie frame preview
- ✍️ Live subtitle editing
- 🔤 Font family selection
- 📏 Adjustable font size
- 🎨 Font color picker
- 🖼️ Upload your own movie frame
- 🔗 Shareable preview links
- 📱 Responsive design
- 🌐 Works directly on GitHub Pages

---

## Why?

Rendering a video just to show a client whether they prefer font size 20 or 22 wastes time.

SubtitlePreview lets both sides see the subtitle style instantly before rendering.
